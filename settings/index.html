Setup Instructions:

  1. Go to Google Cloud Console - Credentials
  2. Create OAuth 2.0 credentials (or use existing)
  3. Enable the Google Assistant API in your project: Enable API
  4. Copy your Client ID and Client Secret below
  5. Click "Get Token" to authorize
Google OAuth2 Configuration

Note: After saving, add the Google Assistant device through the Devices page. The OAuth2 authorization will happen during the pairing process.
